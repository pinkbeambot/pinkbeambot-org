## Step 5: Test Your Agent

Before going live, test your AI:

1. Go to the **Playground** tab in your dashboard
2. Try asking common customer questions:
   - "What are your hours?"
   - "How much does [product] cost?"
   - "What's your return policy?"
   - "How do I contact support?"
3. Review the responses
4. Check the sources (which documents were used)
5. If answers are incorrect or missing, update your knowledge base

### Testing Tips:
- Ask questions in different ways
- Try edge cases and unusual phrasings
- Test with typos and informal language
- Verify citations are correct

**Time:** 5-10 minutes

## Step 6: Deploy to Your Website

Add the chat widget to your site with a simple code snippet:

### For Any Website:
```html
<!-- Add this before the closing </body> tag -->
<script 
  src="https://cdn.pinkbeambot.com/widget.js"
  data-client-id="YOUR_CLIENT_ID"
  data-theme="dark"
></script>
```

### For WordPress:
1. Install the "Pink Beam Bot" plugin
2. Go to Settings â†’ Pink Beam Bot
3. Enter your Client ID
4. Save and activate

### For Shopify:
1. Go to Online Store â†’ Themes
2. Click "Edit Code"
3. Paste the script in `theme.liquid` before `</body>`
4. Save

### For Custom Sites:
Paste the JavaScript snippet anywhere in your HTML before the closing `</body>` tag.

**Time:** 5 minutes

## Quick Reference

### Common Setup Issues:

**Widget not appearing?**
- Check that the script is before `</body>`
- Verify your Client ID is correct
- Clear browser cache and reload

**AI giving wrong answers?**
- Update your knowledge base documents
- Check that sources are being cited correctly
- Adjust the confidence threshold

**Need to pause the AI?**
- Toggle "Offline Mode" in your dashboard
- AI will show a "Back Soon" message
- Resume anytime

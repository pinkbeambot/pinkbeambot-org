## Example 9: Troubleshooting

**Customer:** The chat widget isn't showing on my site.

**AI:** Let's troubleshoot this! Here are common causes:

**Quick Checks:**
1. **Code Placement:** Is the script before the `</body>` tag?
2. **Client ID:** Did you replace "YOUR_CLIENT_ID" with your actual ID?
3. **Cache:** Try clearing your browser cache (Ctrl+Shift+R or Cmd+Shift+R)
4. **JavaScript Errors:** Check browser console (F12) for red errors

**Common Issues:**

**Issue:** "Widget loads but doesn't appear"
- **Fix:** Check if CSS is hiding it (z-index conflicts)
- **Fix:** Ensure the container div isn't set to `display: none`

**Issue:** "Works on desktop but not mobile"
- **Fix:** Check viewport meta tag
- **Fix:** Ensure mobile CSS isn't blocking it

**Issue:** "Shows on some pages but not others"
- **Fix:** Ensure script is on every page template
- **Fix:** Check if different templates use different headers

**Still Not Working?**
Send us:
1. Your website URL
2. Screenshot of where you placed the code
3. Any error messages from browser console

Email: pinkbeambot@gmail.com
Subject: "Widget Not Showing - [Your Website]"

We'll get it fixed within 24 hours!
